** Schema

    | Column     | Type     | Constraints |
    |------------+----------+-------------|
    | first_name | string   |             |
    | last_name  | string   |             |

    | elevator          | boolean  |                       |

*** Images

    | Column      | Type       | Constraints           |
    |-------------+------------+-----------------------|
